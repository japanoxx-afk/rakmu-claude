# rakmu

RhakMu dummy server and binary patch helpers for local restoration testing.

## Main files

- `Start-RhakMuDummyServer.ps1`: TCP dummy server and protocol experiments.
- `Run-RhakMuDummyServer.bat`: launcher for live terminal logging.
- `README_DUMMY_SERVER.md`: current operation notes.
- `Patch-RhakMu*.ps1`: client-side compatibility and crash-guard patch helpers.

Generated logs, packet captures, backups, and local analysis output are ignored.
